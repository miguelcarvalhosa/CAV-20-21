void GolombEncoder::encode ( signed int  value)

A function to encode both signed and unsigned numbers using the Golomb code. The representation of both signed and unsigned numbers is made by adding an additional bit before each quotient representation. Thus a total number of 1 + q + b bits is written in the bitstream to represent each coded value.
