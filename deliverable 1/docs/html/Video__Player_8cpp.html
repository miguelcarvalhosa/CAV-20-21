Detailed Description

Contains the resolution of point 3 of Deliverable 1. This program reads and displays videos of three different formats: 4:4:4, 4:2:2 and 4:2:0.

To use the program, the user must pass five arguments: the input video file path, a running mode, the number of vertical pixels of the video format chosen, the number of horizontal pixels and the format of the video being read.

Usage: point3 <input video="" file>=""> <running mode>="">(optional)

In standard mode, the video runs until the end and afterwards it stops.
