Detailed Description

Contains the resolution of point 5 of Deliverable 1. This program reads a 4:4:4 video file and computes, for each frame, the histogram of three color planes [B,G,R] as well as its grayscale version and its corresponding entropy.

To use the program, the user must pass two arguments: the input video file path and the output histogram count file path

Usage: point5 <input video file> <output histogram file>

At the end the program prints, for each frame, the entropy of three color channels as well as its grayscale version
